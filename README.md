## 🐂 GabiruProMax - App de Formulação de Rações para Bovinos

### Sobre o Projeto

**GabiruProMax** é uma aplicação mobile e web desenvolvida para auxiliar pequenos e médios produtores rurais na formulação de rações e dietas balanceadas para bovinos. O nome homenageia o famoso "Gabiru" - aquele boi macho de rebanho leiteiro que todo produtor conhece e, de alguma forma, se apega!

### 🎯 Objetivo

Democratizar o acesso à nutrição animal de qualidade, permitindo que produtores de todos os portes possam formular dietas econômicas e eficientes sem depender exclusivamente de consultoria especializada.

### ✨ Funcionalidades Principais

- 📊 **Calculadora de Nutrientes**: Balanceamento de proteína, energia, minerais e vitaminas
- 🌾 **Banco de Ingredientes**: Base de dados com valores nutricionais de alimentos disponíveis no Brasil
- 💰 **Custo por Animal**: Cálculo automático do custo da dieta por cabeça/dia
- 📈 **Ganho de Peso Estimado**: Projeção de desempenho baseado na dieta formulada
- 📱 **Modo Offline**: Funciona sem internet no campo
- 🔄 **Salvamento de Receitas**: Biblioteca de formulações salvas
- 📤 **Exportação de Relatórios**: PDF e planilhas para acompanhamento

### 🛠️ Tecnologias

- **Frontend**: React Native / Flutter
- **Backend**: Node.js / Python
- **Banco de Dados**: PostgreSQL / Firebase
- **APIs**: Tabela Brasileira de Composição de Alimentos (TACO)
